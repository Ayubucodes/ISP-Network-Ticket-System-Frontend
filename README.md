# ISP Network Ticket System (Frontend)

This repository contains the frontend application for the ISP Network Ticket System — a modern ticketing platform for Internet Service Providers.

## Overview

- Create, view, and manage support tickets
- Dashboard view for active tickets and queue status
- Role-aware experience (admin vs customer)
- Built with Next.js

## Getting Started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open http://localhost:3000

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
